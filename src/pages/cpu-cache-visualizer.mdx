## Disclosure on the Use of AI/LLM

This project was developed with the assistance of an advanced Large Language Model (LLM) to accelerate code generation, optimize React state management, and assist in designing the UI/UX architecture of the cache visualizer. The LLM was used strictly as a pair-programming tool. All mathematical cache simulations, address partitioning logic, and theoretical concepts presented in this exhibit are derived from our group's primary academic research of the course material, and were thoroughly validated by human review to ensure absolute correctness and alignment with standard Computer Architecture canons.
